##프로미스 체이닝
.then() 메서드를 사용해서 여러 개의 프로미스를 연결하는 것을 말한다.
A프로미스 - return - .then(B프로미스) - return - .then(C프로미스)

.then(result => step1(result))  중복된 result를 줄여서 사용가능
.then(step1)
